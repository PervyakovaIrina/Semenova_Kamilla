**Описание:**
Простое приложение для учёта личных расходов с графическим интерфейсом. Позволяет добавлять расходы с указанием суммы, категории и даты. Реализована фильтрация по категории и подсчёт суммы за выбранный период. Данные сохраняются в JSON-файл.

**Как использовать:**
1. Установите Python и pip.
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите: `python expense_tracker.py`
4. Введите сумму (положительное число), категорию и дату (ГГГГ-ММ-ДД), нажмите «Добавить расход».
5. Для фильтрации — введите категорию в поле фильтра и нажмите «Фильтровать».
6. Для подсчёта суммы — введите даты начала и конца периода и нажмите «Сумма за период».
7. Данные сохраняются автоматически.
